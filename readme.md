# Python for machine learning and data science: NTU PG module

This repository contains the materials for the PSYC40930 module.

The content is organized as follows:

- `python_introduction/`: Introduction to Python for data science.
- `binary_classifiers/`: Materials on binary classification.
- `deep_learning/`: Materials on deep learning.
- `module_details/`: Contains the module schedule, assessment details, and formative assessment information.
